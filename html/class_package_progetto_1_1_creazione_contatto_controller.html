Detailed Description

Controller per l'interfaccia di creazione di un nuovo contatto.

Gestisce l'interfaccia utente e la logica per l'aggiunta di un nuovo contatto. Include il metodo per acquisire i dati dall'interfaccia di creazione contatto per creare un nuovo contatto da salvare nella lista osservabile di Rubrica.

Metodo per salvare un nuovo contatto.

Acquisisce i dati dai textfield, crea un nuovo oggetto Contatto, imposta i rispettivi cambi e lo aggiunge alla rubrica. Viene lanciato cliccando sul pulsante "Salva".

Precondition
L'utente, dopo aver compilato almeno il campo nome o il campo cognome clicca sul tasto "Salva".
Postcondition
Il Contatto viene salvato nella Rubrica.
